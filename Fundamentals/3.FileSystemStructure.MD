## Single Hierarchical Tree Structure
```arudino
/
├─ /bin       (basic binaries)
├─ /etc       (config files)
├─ /home      (user home directories)
├─ /root      (root's home)
├─ /var       (variable files — logs, databases)
├─ /opt       (optional/add-on software)
├─ /mnt, /media (mount points for drives)
├─ /usr       (user programs and libraries)
├─ /tmp       (temporary files)
```
